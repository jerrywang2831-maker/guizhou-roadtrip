# 🚗 武汉→贵州 15天深度自驾环线 · 互动地图

一条从武汉出发、15天深度游贵州黔东+黔南+黔西南+黔北的自驾路线，集景点、美食、住宿于一体。支持**国道优先/高速优先**一键切换。

**5人同行 · 5月30日出发 · 去程一天穿湘西直达凤凰 · 回程经重庆返汉**

## ✨ 功能

- 🗺️ **高德地图交互**：完整行车路线，可缩放、拖动、点击
- 🔀 **路线切换**：国道优先 / 高速优先一键切换，路线信息动态更新
- 📋 **15天行程总览**：点击任一天自动定位到对应地点
- 🏞 **景点+美食+住宿**：每天详细列出，具体酒店名称
- ⌨️ **键盘导航**：← → 方向键切换上/下一天
- 📱 **响应式设计**：桌面端左右分栏，移动端自动适配

## 🚀 快速开始

### 1. 获取高德地图 API Key

访问 [高德开放平台](https://console.amap.com/dev/key/app) 免费申请一个 **JS API** 的 Key。

### 2. 配置 Key

打开 `src/hooks/useAmap.ts`，替换 `AMAP_KEY` 和 `securityJsCode`。

### 3. 开发

```bash
npm install
npm run dev        # 启动开发服务器，默认 http://localhost:5173
npm run build      # 生产构建，输出到 dist/
npm run preview    # 预览生产构建
```

## 🌐 发布到 GitHub Pages

```bash
npm run build
# 将 dist/ 目录部署到 GitHub Pages
```

## 📋 行程总览

| 日期 | 行程 | 亮点 |
|------|------|------|
| Day 1 | 武汉 → 常德 | 桃花源、柳叶湖、常德诗墙 |
| Day 2 | 常德 → 凤凰(半日) → 梵净山 | 沱江+吊脚楼、傍晚到梵净山 |
| Day 3 | 梵净山全天 | 红云金顶、蘑菇石 |
| Day 4 | 梵净山 → **镇远古镇** | 舞阳河S弯全景 |
| Day 5 | 镇远 → 西江苗寨 | 千户苗寨全景日落 |
| Day 6 | 西江 → **荔波小七孔** | 苗岭沿途风光 |
| Day 7 | **荔波全天** | 小七孔古桥、卧龙潭 |
| Day 8 | 荔波 → 贵阳 | 甲秀楼、二七路小吃街 |
| Day 9 | 贵阳全天休整 | 青岩古镇、黔灵山 |
| Day 10 | 贵阳 → 黄果树 | 黄果树大瀑布 |
| Day 11 | 龙宫 + 云峰屯堡 | 水溶洞、明代屯堡 |
| Day 12 | 安顺 → 织金洞 → 遵义 | 溶洞之王、捞沙巷 |
| Day 13 | 遵义 → 茅台 → 赤水 | 茅台酒香、赤水河谷 |
| Day 14 | 赤水丹霞全天 | 十丈洞、佛光岩 |
| Day 15 | 赤水 → 武汉 | 全程高速回家 |

**全程约 3500km · 预算约 ¥3,800/人**

## 🛠 技术栈

- React 19 + TypeScript
- Vite
- 高德地图 JS API v2.0
- Open-Meteo 天气 API
- CSS Modules

## 📝 自定义

如需修改行程，编辑 `src/data/` 中的以下文件：

- `stops.ts`：途经地点及其坐标
- `segments.ts`：相邻地点之间的行车段
- `itinerary.ts`：每天的详细行程（景点、美食、住宿、提示）
